# 🌐 Reposphere
> **The Intelligent Academic Ecosystem: Organize, Search, and Sync with AI.**

Reposphere is a academic resource management platform. Built with a **Neo-Brutalist** aesthetic and a robust **Spring Boot** backend, it transforms traditional file storage into an interactive knowledge base using **Retrieval-Augmented Generation (RAG)**.

## 🎨 Design Philosophy
Reposphere utilizes a **Neo-Brutalist** UI—characterized by bold borders, high-contrast shadows, and raw typography—to provide a distinctive and high-energy user experience that stands out from typical academic tools.

## 📁 Core Features
- **📂 Smart Organization:** Subject-wise categorization with granular folder-level privacy and easy-to-navigate academic materials.
- **📅 Smart Calendar:** Integrated academic calendar for tracking events and managing semester schedules.
- **☁️ Cloud Storage:** High-availability file and asset hosting integrated with **Cloudinary** for seamless media management.
- **🔐 Secure Access:** Protected document management ensuring your resources are only accessible by intended users.

## 🛠️ Tech Stack

| Layer | Technology |
| :--- | :--- |
| **Frontend** | React.js, Tailwind CSS (Neo-Brutalist) |
| **Backend** | Java Spring Boot  |
| **Database** | PostgreSQL (Supabase) |
| **Storage** | Cloudinary |
| **AI/ML** | Retrieval-Augmented Generation (RAG) |
